# Dothraki AGOB Addon

A small example addon for `A Game of Blocks` that adds a custom Dothraki kingdom, custom Dothraki houses, a custom religion, custom troops, a quest, and a themed blacksmith content set.

## Features

- Play as a custom `Dothraki` kingdom in Essos
- Choose between three Dothraki house options:
  - Khal Drogo
  - Khal Jhaqo
  - Khal Pono
- Follow the custom `Great Shepherd` religion
- Spawn in the Dothraki Sea or Dothraki Sea Steppe
- Recruit a custom Dothraki rider troop
- Try a sample Dothraki quest
- Use custom Dothraki armor and blacksmith recipes

## What This Addon Is

This project is mainly a clean example addon for AGOB. It is meant to show addon developers how custom kingdoms, troops, quests, textures, and other content can be added through the public addon API.

That means it is intentionally small, lightweight, and easy to copy from.

## Requirements

- Minecraft `1.21.1`
- Fabric Loader
- Fabric API
- `A Game of Blocks`

## Notes

- This addon is designed as an example project first.
- Some content is intentionally lightweight or placeholder-style so developers can expand it for their own projects.
- The blacksmith addon tab example requires an AGOB version that includes the matching addon blacksmith API support.
